Class: Bluetooth

Bluetooth

new Bluetooth()

Bluetooth class include all useful bluetooth global interfaces.

Please note that the application should not create Bluetooth object, BC manages the object model.

Properties:
Name Type Description
devices Array.<Device> The advertising devices, this field filled after 'BC.Blueooth.StartScan' be called
